#  ASP.NET Core 9.0 MVC Portfolio Management System

A comprehensive and modern **Portfolio Management System** built with the latest **.NET 9.0**.  
Includes both a **personal portfolio website** and an **admin panel** for full content management.  

---

##  Project Overview

This project is designed for developers, freelancers, and creative professionals who need a professional online portfolio.  
It provides two distinct user experiences: a **portfolio showcase for visitors** and an **admin panel for content management**.  

- 🔹 **Dual-purpose system** → Portfolio website + Admin panel  
- 🔹 **No-code content management** → Manage all content with CRUD operations  
- 🔹 **Modern .NET 9.0** → Latest framework features  
- 🔹 **Responsive design** → Fully mobile-friendly with Bootstrap 5  

---

##  Technical Details

### ⚙️ Technologies Used
- ASP.NET Core **9.0 MVC**  
- Entity Framework Core **9.0.8** (SQL Server)  
- Bootstrap 5 (Responsive UI)  
- MVC Pattern  
- View Components  

### Database Entities
- **About** → Personal information  
- **Contact** → Contact details  
- **Experience** → Work experiences  
- **Feature** → Highlighted features  
- **Message** → Visitor messages  
- **Portfolio** → Projects  
- **Skill** → Skills & proficiency levels  
- **SocialMedia** → Social media links  
- **Testimonial** → References  
- **ToDoList** → Task management  

---

##  Controllers & Features

- **DefaultController** → Main portfolio view (visitor side)  
- **ExperienceController** → Full CRUD operations for experiences  
- **MessageController** → Manage messages (inbox, read/unread, details)  
- **ToDoListController** → Task management (add, edit, toggle status)  
- **StatisticController** → Dashboard statistics (skills, messages, unread counts)  

---

##  UI & View Architecture

###  User Interfaces
- **Portfolio Website** → `/Default/Index`  
- **Admin Panel** → `/Layout/Index`  

###  View Components
- **Portfolio Side** → `_About`, `_Contact`, `_Experience`, `_Feature`, `_Portfolio`, `_Skill`, `_Testimonial`, `_Statistic`, `_Navbar`, `_Footer`  
- **Admin Panel** → `_LayoutHead`, `_LayoutNavbar`, `_LayoutSidebar`, `_LayoutScript`  

---

## Key Features

✅ Dual-purpose system: Portfolio + Admin panel

✅ Responsive mobile-friendly design
